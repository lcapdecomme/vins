# Gestion de cave à vins / Wine Cellar Application #
Démo here : *http://macave.byethost10.com*


## Set Up

1. Execute vins.sql (directory sql) to create a neuw database vins and populate the "wine" tables:
   ** mysql vins -uroot < vins.sql
2. Deploy sources in your httpd server
